# eider: Declarative Feature Extraction from Tabular Data Records

`eider` is a lightweight package for processing tabular data in a
declarative fashion. To get started, see:
[`vignette("eider")`](https://alan-turing-institute.github.io/eider/docs/articles/eider.md)
